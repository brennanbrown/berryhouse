## Building This Site: By the Numbers

I practice what I preach. This site is built with the same tools and principles I offer clients:

### Tech Stack
- **Static Site Generator**: Eleventy 3.0 (blazing fast, zero client-side JavaScript for core functionality)
- **Styling**: Tailwind CSS with custom design system
- **Typography**: Google Fonts‚ÄîSpace Grotesk for headers, Public Sans for body
- **Hosting**: Netlify (JAMstack-optimized, global CDN)
- **Version Control**: Git + GitHub (transparent, collaborative workflow)

### Performance & Accessibility
- **Build time**: ~8.5 seconds for 67 pages
- **Lighthouse scores**: 100/100 across the board (performance, accessibility, best practices, SEO)
- **Web Vitals**: All metrics in the green
- **Accessibility features**: 
  - Reduce motion toggle (respects `prefers-reduced-motion`)
  - High contrast mode
  - Font size controls
  - System fonts toggle
  - Semantic HTML throughout
  - WCAG 2.1 AA compliant

### IndieWeb Ready
- **Microformats2**: h-card, h-entry for machine-readable content
- **Webmentions**: Integrated with webmention.io
- **RSS/Atom feed**: Full-text feed at `/feed.xml`
- **No tracking**: Privacy-first analytics with Plausible (no cookies, aggregated data only)

### The Build Process
Just as an example, my last feature update for this site included:

- Generated **11 icon formats** from a single logo (favicons, Apple touch icons, PWA icons, OG images)
- Created **custom animations** that respect accessibility preferences
- Integrated **Google Fonts** with fallback to system fonts
- Built a **complete icon generation pipeline** for future updates
- Implemented **SEO optimization** (robots.txt, sitemap.xml, structured data)
- Set up **GitHub Sponsors integration** with FUNDING.yml

Everything is version-controlled, documented, and designed to be maintainable long-term. No proprietary platforms. No vendor lock-in. Just clean, portable code you can own.

A free, searchable directory of over 150 grassroots organizations, clubs, activist groups, and community initiatives in Calgary. Built to help people find their community and connect with local organizing efforts.

Key features:
- **Advanced search & filtering** by name, organization type, interests, meeting format, and location
- **Rich taxonomy** with categories for Nonprofits, Grassroots groups, Social Clubs, Chapters, and more
- **Fully responsive** design that works seamlessly on all devices
- **Dark mode support** with custom color palette
- **PWA capabilities** - installable with offline support
- **Open source** with data stored as markdown files in Git
- **Fast & lightweight** static site with no database required
- **Community-editable** through GitHub contributions

The project prioritizes groups that are community-led, center marginalized voices, and focus on local organizing and mutual aid work.
